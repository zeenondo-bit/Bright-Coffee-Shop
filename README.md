# Bright-Coffee-Shop Sales Analysis

**Explain what the project is about**
This project explores sales data from Bright Coffee Shop in order to understand how different products amd time of day patterns contribute to overall business performance. It includes an analysis of revenue, product sales volumes and customer purchasing behaviour, supported by visualisations and insights that highlight which products drive the most value and when  the store performs at its peak

**Objective of the project:**
The goal of this analysis is to identify key sales trends and uncover opportunities to improve operational efficiency, product stragegy and revenue growth. By examining product performance and time-based sales patterns, the project provides clear recommendations to help Bright Coffee Shop optimise its offerings, strenghthen peak trading periods and address underperfoming areas
